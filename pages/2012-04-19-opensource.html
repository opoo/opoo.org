OpooPress 是博主自行开发的一个静态博客生成器，使用 Java 语言和 FreeMarker 模板开发 生成器 部分，默认带一个支持 CSS3 响应式设计的主题。。

Java Object Query System

一个Java应用。JOQS是一个超轻量级的ORM产品，一个JDBC的扩展应用。

SourceForge Project Web Email Configuration

一个WordPress插件。解决解决在SourceForge空间安装WordPress无法发送邮件的问题。
